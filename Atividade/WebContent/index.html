Passos para utilizar o Git Hub

Depois de criar seu usuario e acessa-lo

No Eclipse

  • Vai em window depois em preferences e entra na parte do "GIT"
  • Escolha qual pasta ele irá clonar o projeto
  • Depois vamos adiconá-lo na barra de tarefa
  • Configuramos também as senhas e agora iremos conectar com o eclipse com o git
  • Acessar o site do Git hub
  • Criar um repositório no github
  • Clonar o repositório
  • Realizar o comit(envio) de change para o git HUB
